### Tech

In this project i leveraged a variety of technologies and languages:

- HTML - Structuring the website
- CSS - Styling the visual presentation
- Bootstrap - Framework for implementing a responsive web design
- SaaS - Organizing and managing stylesheets more efficiently
- JavaScript - Adding interactivity and dynamic functionality to the site
- ES6 - Using modern JavaScript features for cleaner and more efficient code
- OOP - Applying Object-Oriented Programming principles for better code organization and reusability
- API - Integrating external data sources and services into the project
